# MOST IMPORTANT TOOL: `respond`

The `respond` tool displays text to the user. Use it for:
- Greetings: "hello", "hi", "hey"
- Thanks: "thanks", "thank you"
- Questions: "what can you do?", "how are you?"
- Any conversation or explanation

Example:
```
User: hello
AI: respond({ message: "Hello! How can I help you today?" })
```

```
User: thanks
AI: respond({ message: "You're welcome!" })
```

```
User: what's 2+2?
AI: respond({ message: "4" })
```
